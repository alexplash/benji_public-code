# Benji — Autonomous Patrol & Conversational Robot

(images/benji.jpg)

Autonomous patrol + conversational robot, using a self-hosted GPT language model, Whisper STT, and ElevenLabs TTS. Implements SLAM-style navigation with ultrasonic sensors, multi-user identity memory, agentic movement/vision commands, and a custom Q-table RL loop with sentiment analysis to adapt Benji’s behavior per user.
